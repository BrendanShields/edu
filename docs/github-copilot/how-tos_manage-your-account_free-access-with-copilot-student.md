# Access GitHub Copilot for free as a student

Learn how to access Copilot's premium features for free as a student.

Copy as Markdown

## In this article

## [About Copilot Student access](#about-copilot-student-access)

Verified students on GitHub Education are eligible for **free access to Copilot's premium features**. See [About individual GitHub Copilot plans and benefits](/en/copilot/concepts/billing/individual-plans#github-copilot-student).

To learn about becoming a verified student, see [Apply to GitHub Education as a student](/en/education/explore-the-benefits-of-teaching-and-learning-with-github-education/github-education-for-students/apply-to-github-education-as-a-student).

GitHub reevaluates your eligibility every month.

## [Accessing Copilot Student](#accessing-copilot-student)

1.  Navigate to [https://github.com/settings/education/benefits](https://github.com/settings/education/benefits).
    
    If you're not yet a verified student, you'll need to click **Start an application** and complete the verification process.
    
2.  Under "Free GitHub developer resources for students and teachers", click **Learn more**.
    
3.  Follow the prompts to activate Copilot Student, configuring the Copilot use policies to meet your needs.
