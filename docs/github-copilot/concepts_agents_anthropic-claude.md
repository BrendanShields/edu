## [Introduction](#introduction)

The Anthropic Claude coding agent uses the Claude Agent SDK and can be powered by your existing Copilot subscription. For information about Anthropic Claude, see the [Anthropic Claude documentation](https://platform.claude.com/docs/en/agent-sdk/overview).

## [Anthropic Claude coding agent](#anthropic-claude-coding-agent)

Before you can assign tasks to Anthropic Claude coding agent, it must be enabled. See [Managing GitHub Copilot policies as an individual subscriber](/en/copilot/how-tos/manage-your-account/manage-policies#enabling-or-disabling-third-party-agents-in-your-repositories).

To learn more about using third-party coding agents, see [About third-party agents](/en/copilot/concepts/agents/about-third-party-agents).
