# Canceling GitHub Copilot for your enterprise

Disabling GitHub Copilot for all organizations in your enterprise will cancel your enterprise's Copilot plan.

## Who can use this feature?

Enterprise admins

Copilot Enterprise or Copilot Business

Copy as Markdown

To cancel your GitHub Copilot plan for your enterprise account, you need to disable GitHub Copilot for all organizations in your enterprise. See [Disabling Copilot for organizations in your enterprise](/en/copilot/how-tos/administer-copilot/manage-for-enterprise/manage-access/disable-for-organizations).
